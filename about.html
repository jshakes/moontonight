About

The Moon Tonight is an experiment by James Shakespeare in using CSS to render the moon in all its possible phases.

The code is available to view on Github.

Credits

Moon data and weather forecast from The Dark Sky Forecast API.

Location autocomplete data from Google Places Autocomplete.

Satellite dish icon (used in the loading animation) originally by Arthur Shlain from the Noun Project.

Cloud icon (used in the cloud cover warning icon) originally by Edward Boatman from the Noun Project.

Font used is Reader by Colophon.
